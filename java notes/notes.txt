32. Streams in Java8 Collections in Java - Part7
    - What is a Stream?
    - Stream vs Collection
    - Stream Creation
    - Intermediate Operations
    - Terminal Operations
    - Lazy Evaluation
    - Processing Order
    - Single Use
    - Parallel Streams

    Summary : 
        Stream	            Pipeline:create → intermediate ops → terminal op
        Intermediate	    Lazy — filter, map, flatMap, sorted, distinct, peek, limit, skip
        Terminal	        Triggers execution — forEach, collect, reduce, count, min/max, findFirst
        Lazy evaluation	    Nothing runs until terminal op is invoked
        Processing order	Element-by-element through pipeline (except stateful ops like sorted)
        Single use	        Stream consumed after terminal op — create new for reuse
        Parallel	        parallelStream() — fork-join pool — faster for large data
